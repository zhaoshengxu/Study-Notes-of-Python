#### 三.变量与赋值

##### 1. 需要牢记的三点
- 1. 一切数据都是对象
- 2. 一切变量都是对数据对象的引用
- 3. Python内部的引用计数：sys.getrefcount()

---

##### 2. 变量命名规范
- 1. **记住不能做什么**：不能用关键字，开头不能用数字,不能用操作符。
- 2. **怎么搭配最合理**：大小写敏感，尽量使用小写字母，没有特殊情况少用大写；清晰；尽量不要用拼音，如果用的话，用全称。

---

##### 3. 赋值
- 1. 记住动态特性：在赋值时确定数据类型

```
a = 1         # int
b = 'hello'   # string
c = [1,2,3]   # list
```

- 2. 多重赋值
- 3. 删除
- 
```
a,b,c = 'how', 'are', 'you'
del a,b,c
```
